# sk-demo
 Semantic Kernel Demo and Samples

## Setup

Add appsettings.json with:

{
  "BingSearch": {
    "ApiKey": "YOUR_BING_SEARCH_API_KEY"
  },
  "OpenAI": {
    "ApiKey": "YOUR_OPENAI_API_KEY",
    "Endpoint": "YOUR_OPENAI_ENDPOINT",
    "DeploymentName": "gpt-4o-mini"
  }
}

## Run

dotnet run sk-demo.cs
